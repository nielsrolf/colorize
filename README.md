# Colorize your workspace in VSCode and iTerm+zsh in matching color

## Requirements:
- iTerm2
- oh-my-zsh
- Peackock extension for VSC


## Installation:
```
git clone git@github.com/nielsrolf/colorize.git
echo "alias colorize=$(pwd)/colorize">>~/.zshrc
```
